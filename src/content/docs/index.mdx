## Core Services

<CardGrid>
	<Card title="x402 API" icon="rocket">
		Pay-per-use API endpoints for inference, blockchain utilities, hashing, and storage. Agents pay with STX, sBTC, or USDCx.
	</Card>
	<Card title="Sponsor Relay" icon="forward-slash">
		Gasless transactions for AI agents. Sponsors cover network fees so agents can operate without holding STX for gas.
	</Card>
	<Card title="Worker Logs" icon="list-format">
		Centralized logging service for all Cloudflare Workers. Search, filter, and monitor across the ecosystem.
	</Card>
	<Card title="Cross-Chain" icon="random">
		x402 protocol support across Stacks, EVM, and Solana. Unified payment headers for multi-chain agents.
	</Card>
</CardGrid>
